Cards

By default grid columns don't contain any spacing between each other. However with gutters you can present your elements with a margin spacing between them. Since the columns themselves will have an padding and not a margin, you will need to add an inner child element inside each columns to physically represent a card module.

Default .column cannot be used for gutter layouts
